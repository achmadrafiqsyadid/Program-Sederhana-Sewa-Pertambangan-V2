Rancang Bangun Pengolahan Data Penyewaan Alat Berat
Deskripsi

Repository ini berisi sebuah aplikasi pengolahan data penyewaan alat berat yang dibuat untuk membantu proses pencatatan dan pengelolaan data operasional secara lebih rapi dan terstruktur.

Aplikasi ini menangani alur penyewaan mulai dari pendataan konsumen, alat berat, dan operator, hingga proses transaksi penyewaan, pengembalian alat, pembayaran, serta pembuatan laporan. Seluruh data saling terhubung sehingga dapat mengurangi penginputan berulang dan meminimalkan kesalahan pencatatan.

Proyek ini dikembangkan sebagai tugas mata kuliah Visual 3, dengan fokus pada penerapan visual programming dan perancangan antarmuka yang mudah digunakan.

Latar Belakang

Pada proses penyewaan alat berat, pencatatan manual sering menimbulkan kendala seperti data yang tidak konsisten, sulit ditelusuri, dan lambat saat dibutuhkan laporan.
Aplikasi ini dibuat sebagai solusi sederhana untuk membantu pengelolaan data penyewaan agar lebih tertata dan mudah dipantau.

Fitur Aplikasi

Beberapa fitur utama yang tersedia dalam aplikasi ini antara lain:

Manajemen Konsumen
Menyimpan data perusahaan atau pelanggan penyewa alat berat.

Manajemen Operator
Mengelola data operator beserta informasi lisensi dan status ketersediaan.

Manajemen Alat Berat
Mengatur data unit alat berat, termasuk jenis, spesifikasi, tarif sewa, dan status alat.

Transaksi Penyewaan
Mencatat proses penyewaan alat berat berdasarkan konsumen, alat, operator, lokasi proyek, dan periode sewa.

Pengembalian Alat
Mencatat kondisi alat saat dikembalikan serta denda jika terjadi kerusakan atau keterlambatan.

Pembayaran Sewa
Mencatat pembayaran dan status pelunasan transaksi penyewaan.

Laporan
Menyediakan laporan penyewaan dan pembayaran berdasarkan rentang tanggal tertentu.

Alur Penggunaan Singkat

Input data master (konsumen, operator, dan alat berat)

Melakukan transaksi penyewaan alat berat

Proses pengembalian alat dan pencatatan kondisi

Melakukan pembayaran sewa

Menampilkan dan mencetak laporan

Teknologi & Konsep

Pengolahan data transaksi dan laporan
